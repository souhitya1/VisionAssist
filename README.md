# VisionAssist AI Glasses

VisionAssist AI Glasses is an AI-powered assistive system designed to help visually impaired individuals navigate safely and interact with their surroundings using real-time audio feedback.

## 🚀 Current Progress (Prototype Stage)

We have successfully developed and demonstrated the following components:

### 🧩 CAD Design (Hardware Prototype)
- Designed a wearable smart glasses structure using CAD
- Includes placement planning for:
  - Camera module
  - Microphone
  - Audio output (bone conduction/earphones)
- Focused on comfort, usability, and real-world deployment

### 👨‍👩‍👧 Face Recognition System (Working Module)
- Implemented a system to recognize known individuals
- User can upload images of family members
- The system detects and identifies the person in real-time
- Outputs audio feedback such as:
  - “Father is in front of you”
